# Digital Forensic Investigation & AI Pattern Recognition
**Researcher:** Balogun Olajide Olufemi  
**Affiliation:** Nigerian Defence Academy (NDA), Dept. of Cyber Security  

## Project Overview
This repository explores the application of Artificial Intelligence in identifying illicit drug trafficking patterns across Social Media and Dark Web environments. My MSc thesis focuses on the forensic extraction of intelligence to disrupt criminal networks.

## The AI Intersection
In this project, I apply Machine Learning to automate the "Pattern Recognition" phase of digital forensics:
* **Natural Language Processing (NLP):** Using **Hugging Face** transformers to detect "coded" language used in illicit transactions.
* **Predictive Analysis:** Utilizing **Scikit-learn** to build classification models that distinguish between benign social activity and organized trafficking signatures.
* **Data Synthesis:** Leveraging **Pandas/NumPy** to aggregate and clean unstructured data from onion sites and social platforms.

## Tech Stack
* **Language:** Python
* **AI Frameworks:** Scikit-learn, Pandas, NumPy
* **Forensic Focus:** OSINT, Dark Web Scraping, Cryptographic Analysis
